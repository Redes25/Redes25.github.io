HTML:


Lenguaje de marcado de hipertexto o HyperText Markup Language por sus siglas en inglés. Es un lenguaje que permite definir y darle estructura a las páginas web.

BODY:


Indica la parte del cuerpo del contenido de un documento HTML.

CENTER:


Ubicación física que almacena máquinas de computación y sus equipos de hardware relacionados.

H1:


Es una etiqueta HTML que se utiliza para indicar cuál es el título del contenido de un sitio web.

HR:


Representa un cambio de tema entre párrafos.

BR:


Es útil para escribir un poema o una dirección, donde la división de las líneas es significante.

P:


Es el apropiado para distribuir el texto en párrafos.

HEAD:


Es la primera de las dos partes en que se estructura un documento HTML.

TITLE:


Es un tag de código HTML que le permite darle un título a una página web.
